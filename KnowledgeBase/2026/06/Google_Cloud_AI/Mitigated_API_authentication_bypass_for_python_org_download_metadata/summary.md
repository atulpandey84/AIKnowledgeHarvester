On February 23rd 2026, Splitline Ng from the DEVCORE Research Team reported to the Python Security Response Team (PSRT) an authentication bypass vulnerability in the python. org release management API. By supplying an admin username with an arbitrary API key the request was processed with admin privileges. If exploited, this would have allowed an attacker to modify Python release and file metadata that affects what URLs users are offered when visiting python.
